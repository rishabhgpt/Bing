Welcome to multiwall.

To retrieve your wallpapers from momentumdash app :

$ git clone http://github.com/rishabhgpt
$ cd multiwall
$ chmod 755 momentum imgur
$ crontab -e

Now add momentum script as daily cronjob


$ 0 10 * * * /path/to/multiwall/momentum

Similar process for imgur script
